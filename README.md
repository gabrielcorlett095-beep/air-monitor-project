# Monitor de Qualidade do Ar Pro
Este projeto é uma aplicação web desenvolvida em Python com Streamlit para monitoramento de qualidade do ar em tempo real, utilizando dados da API Open-Meteo.

## Tecnologias Utilizadas
- **Linguagem:** Python 3.12
- **Framework:** Streamlit
- **Banco de Dados:** PostgreSQL (hospedado no Render)
- **Deploy:** Streamlit Community Cloud

## Funcionalidades
- Consulta de poluição (PM2.5, NO₂, O₃) por cidade.
- Histórico de pesquisas persistido em banco de dados (CRUD).
- Visualização de dados via gráficos e mapas interativos.
